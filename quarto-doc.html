파이썬 코드 청크

파이썬 코드 청크는 다음과 같은 단축키를 통해서 넣을 수 있음.

  • 단축키: Ctrl + Alt + I

코드 청크 종류

2가지 종류가 있음

  1. 인라인 코드청크: 백틱 하나(`)로 묶음
  2. 디스플레이 코드청크: 백틱(```) 세 개로 묶음

실행가능한 코드 청크

백틱 뒤에 {python}을 붙여주면 됨.

옵션은 #|로 시작함.

  • echo 옵션: 코드를 문서에 보여줌
  • eval 옵션: 코드를 돌릴 것인가, 그냥 둘 것인가를 결정

수식 입력 방법

  • 인라인 수식: $ 하나로 감싸준다. (본문에 입력)
  • 디스플레이 수식: $ 두개로 감싸준다. (독립적인 문단)

문장안에 숫자 \(5^2\)를 넣어보자.

여러숫자들을 따로 떼어 표시하자.

\[ \sqrt[5]{2}^{3} \]
